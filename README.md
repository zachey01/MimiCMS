# MimiCMS <img src="https://zachey01.github.io/MimiCMS/logo.svg" height="40">

> The CMS for Valve Game Servers!

#### Features:

-   Backups
-   Admin panel
-   RCON
-   Shop
-   Tickets
-   Steam auth (openid)
-   Custom pay system
-   Very fast
-   Simple installation

### Installation

Go to website http://mimicms.zachey.space/#install and fill out form and click the "Download" button.
